## Alethio project

---

### 적용 기술

- React.js (functional), React Hooks, Redux-saga, Styled components

### 구현 페이지

- Header (Menu)
- Main (Service)
- Signup
- Login
- Mypage
- MypageDetail

### 구현 기능

- Header 반응형 레이아웃
- Nav 선택시 색 변화
- 회원가입, 로그인 validation
- 리덕스 사가를 이용한 회원가입 토큰 관리
- RESTful API 이용하여 아이템 목록 구현
- async axios await 구현하여 비동기 처리, 로딩화면 구현
